# 莱安家装案例分享微信小程序

这是一个从零搭建的微信小程序 MVP，围绕“设计师分享单个案例给客户查看”这一核心场景实现。

## 当前能力

- 设计师账号登录
- 首页、案例列表、案例详情
- 案例新增/编辑/删除
- 固定字段录入
- 自定义内容模块
- 单案例分享
- 客户只能查看分享案例详情
- 无权限拦截页

## 运行方式

1. 用微信开发者工具打开当前目录
2. 选择“小程序项目”
3. `appid` 可先使用测试号或开发号
4. 编译后进入登录页
